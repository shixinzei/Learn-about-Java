1、构造器的声明格式，构造器的特点？
构造器的声明格式：
权限修饰符 类名(形参列表){
	
}

构造器的特点:
(1)一个类可以有多个构造器，构成重载
(2)所有的Java类都有构造器
一个类如果没有声明任何构造器，编译器会自动增加一个默认的无参构造
(3)一个类如果声明了其他构造器，编译器就不会增加默认的无参构造
(4)构造器名称必须与类名相同

2、this关键字代表什么？this关键字的用法？
this关键字代表当前对象
（1）在构造器中出现this，表示正在创建的对象
（2）在成员方法中出现this,表示当前对象，调用该方法的对象

this关键字的用法：
（1）this.属性
当局部变量与属性重名时，在属性前面加this.
（2）在方法中，如果需要返回当前对象
return this;
（3）this.方法
表示调用当前对象的其他成员方法，可以省略this.
（4）this()或this(实参列表)
表示调用本类的其他的构造器
this()表示调用无参构造
this(实参列表)表示调用有参构造
this()或this(实参列表)必须在构造器的首行，而且要注意避免递归调用

3、如何声明包？声明包的语句有什么要求？
如何声明包？
package 包名;

声明包的语句有什么要求：必须在源文件的首行

4、如何使用其他包的类？其他包的类要能够被使用有什么要求？
如何使用其他包的类：
（1）使用类的全名称
包.类名 
java.util.Scanner input = new java.util.Scanner(System.in);
（2）使用导包语句 + 简名称
import 包名.类名;
Scanner input  = new Scanner(System.in);

其他包的类要能够被使用有什么要求：
（1）必须在同一个项目中
（2）其他包的类必须可见的
